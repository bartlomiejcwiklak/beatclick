### About

beatclick is a browser drum-grid sequencer

### How to run

1. Install dependencies:

```bash
npm install
```

2. Start dev server:

```bash
npm run dev
```

### Controls

- Space: play/stop
- 1, 2, 3, 4: preview the four loaded sample slots
