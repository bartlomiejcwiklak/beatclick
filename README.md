is a browser drum-grid sequencer.


### how to run

you can use the pre-built demo deployment [here](https://bartlomiejcwiklak.github.io/beatclick/) to skip all the hassle

but, if you'd rather not do that, then:

```bash
npm install && npm run dev
```
will do the trick just as well.


### key features

- **pro audio engine**: high-performance web audio api scheduler for rock-solid timing.
- **multi-kit presets**: swap between **trap, house**, and more curated drum kits on the fly.
- **swing & groove**: adjustable swing control to give your beats a human, off-the-grid feel.
- **real-time control**: smooth bpm adjustments and instant pattern updates without audio dropouts.
- **interactive sequencer**: a tactile 16-step grid with active playhead tracking and one-shot previews.

### tech stack

- **core:** react 19 + typescript
- **bundler:** vite
- **styling:** tailwind css 4
- **audio engine:** custom react hooks + web audio api
- **assets:** curated high-quality drum samples
